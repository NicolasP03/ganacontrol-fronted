# GanaControl - Frontend

Aplicación web para la gestión de fincas ganaderas, desarrollada con React. Permite interactuar con el sistema para administrar procesos como ganado, ventas, salud y reproducción, consumiendo una API REST.

## Tecnologías
- React
- JavaScript
- CSS
- Axios

## Funcionalidades
- Visualización de información del ganado
- Registro y consulta de ventas
- Gestión de salud animal
- Módulo de reproducción
- Gestión de inventario
- Consumo de API REST
- Navegación entre módulos

## Instalación
```bash
npm install
npm start
```

## Estructura
- components → componentes reutilizables
- pages → vistas principales
- services → conexión con la API
- assets → recursos gráficos
